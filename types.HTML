Types of operating systems

  1. Batch Operating System- Sequence of jobs in a program on a computer without manual interventions
  2. Time sharing operating System- allows many users to share the computer resources.(Max utilization of the resources).
  3. Distributed operating System- Manages a group of different computers and make appear to be a single computer.
  4. Network operating system- computers running in different operating system can participate in common network (It is used for security purpose).
